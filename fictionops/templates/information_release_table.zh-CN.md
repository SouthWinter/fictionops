# 信息释放表

> 用于回答：谁知道什么？以什么版本知道？这个信息下一次应该如何扩散？

| 信息/秘密 | 作者真相 | 读者目前知道 | 角色/群体 A 知道 | 角色/群体 B 知道 | 公共版本 | 官方版本 | 下一次释放 | 禁止事项 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |  |  |

## 使用原则

1. 作者真相不等于角色认知。
2. 公共谣言可以是错的，但要有传播路径。
3. 官方版本可以掩盖事实，但要服务权力。
4. 读者知道的信息不等于所有角色知道。
5. 如果某个角色突然知道太多，必须有来源、证据或误读过程。
