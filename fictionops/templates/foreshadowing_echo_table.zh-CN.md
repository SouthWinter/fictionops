# 伏笔回声表

> 用于追踪伏笔如何被读者记住，而不是每次都被旁白解释。

| 线程 | 初次埋下 | 上次回声 | 当前读者记忆 | 下一次轻回声 | 禁止提前解释 | 兑现/转化方向 |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |

## 回声方式库

- 物件转手；
- 同一动作换语境；
- 旧话改一字；
- 谣言跨地区变形；
- 身体反应；
- 角色误用旧信息；
- 禁忌话题周围的沉默；
- 小人物只知道一半；
- 场景结构重复但结果相反。

## 审计问题

- 这条伏笔是否太久没有出现？
- 本次是否解释过多？
- 读者需要的是提醒，还是需要新信息？
- 这条线是否被角色提前说穿？
- 兑现时是否会像临时补丁？
