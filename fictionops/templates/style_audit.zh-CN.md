## 1.5 感官语义场

> 不只统计单字，要按意象家族判断是否重复承担同一种心理功能。核心意象可以高频，但必须有功能，不应变成情绪按钮。

| 语义场 | 代表词 | 主要功能 | 是否被用作心理代词 | 处理建议 |
| --- | --- | --- | --- | --- |
| 冷系 | 冷 / 寒 / 冰 / 雪 / 冻 / 凉 | 环境 / 规则 / 威胁 / 身体后果 |  |  |
| 热系 | 热 / 暖 / 烫 / 烧 / 火 / 温 | 生活 / 亲近 / 血温 / 羞惭 / 急迫 |  |  |
| 灯暗系 | 灯 / 光 / 亮 / 暗 / 影 | 视线 / 权力 / 仪式 / 信息遮蔽 |  |  |
| 血白灰系 | 血 / 白 / 灰 / 黑 | 神话 / 死亡 / 文书 / 旧案痕迹 |  |  |
| 场景专属物象 | 如门 / 线 / 格 / 账 / 缆 / 灶 / 绳 | 角色视角材料 / 地域气味 |  |  |

- 先标注每处感官词的功能：环境、物件规则、伏笔回声、身体反应、心理代词。
- 若同一章里同一语义场连续三次承担同一种心理，例如恐惧都写冷、亲近都写暖、压迫都写暗，优先替换其中一两处。
- 替换时不要机械删词，可改成动作、重量、声音、空间距离、时间错位、职业材料或角色特有物件。
- 判断“主题性重复”和“模板性重复”：前者让章节意象变厚，后者让读者感觉作者在重复按按钮。
